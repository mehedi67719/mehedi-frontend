# PPA - Master Focus & Productivity Landing Page

A high-performance, professional landing page built to help users master deep work and eliminate procrastination. This project features a sleek, dark-themed UI with custom responsiveness.

## 🌐 Live Demo
Check out the live site here:  
**[Live Preview](https://mehedi-frontend.vercel.app/)** 

## ✨ Key Features

* **Next.js 16 Power:** Utilizing the latest Next.js features for optimized performance and routing.
* **Custom Responsive Design:** Tailored breakpoints to ensure a pixel-perfect experience across all devices.
* **Modern Dark UI:** A minimalist, "dark moody" aesthetic styled with Tailwind CSS and DaisyUI.
* **Interactive Components:** Integrated video previews and conversion-focused Call-to-Action (CTA) sections.
* **Productivity Timer:** Dynamic countdown banner to create urgency for course enrollment.

## 🛠️ Tech Stack

* **Framework:** Next.js 16 (App Router)
* **Styling:** Tailwind CSS & DaisyUI
* **Icons:** Lucide React
* **Font:** Geist (Optimized via `next/font`)

## 📐 Custom Breakpoints
This project uses a custom Tailwind configuration to match specific design requirements:
* **XL (Desktop):** 981px+
* **LG (Tablet LG):** 681px - 980px
* **MD (Tablet):** 521px - 680px
* **SM (Mobile):** 412px - 520px
